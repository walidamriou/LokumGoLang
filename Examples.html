Examples
Any example can use it by this method, the comments in the code will help you:

- To run the program use go run:
$ go run main.go
hello to world of Go Language!
You know that 1+1 is: 2
- To build it to binaries. use go build:
$ go build main.go
$ ls
main main.go

- To execute the built binary after build it use ./helloworld:
$ ./main
hello to world of Go Language!
You know that 1+1 is: 2



Note: you can use https://play.golang.org to test the code,
and "Read it" open the file in the browser but only in Chromium based browsers
